# 上市公司制造业企业AI应用评估项目

## 项目概述

本项目基于BERT模型，对上市公司制造业企业的AI应用情况进行评估。通过分析企业年报中的文本信息，自动识别企业是否使用AI技术、AI使用方式以及AI应用类型。

## 数据说明

项目使用 `合并结果.xlsx` 文件作为训练数据，包含以下字段：
- 股票代码：上市公司股票代码
- 年份：数据所属年份
- 句子：从年报中提取的句子
- 所属年报总句子数：该年报的总句子数
- 是否存在AI应用：标记句子是否提及AI应用
- AI使用方式：AI的使用方式（如自身应用、产品输出等）
- AI应用类型：AI的应用类型（如生产型、决策型等）

## 环境搭建

1. 安装依赖包

```bash
pip install -r requirements.txt
```

2. 确保有足够的计算资源（建议使用GPU进行训练）

```
.

├── predict.py          # 预测脚本
```

## 使用方法

### 1. 训练模型

```bash
python train.py
```

训练过程中会自动保存模型到 `models` 目录。

### 2. 评估模型

```bash
python evaluate.py
```

评估结果会显示三个任务的准确率、精确率、召回率和F1分数。

### 3. 预测新文本

```bash
python predict.py
```

默认使用示例文本进行预测，可在 `predict.py` 文件中修改 `sample_texts` 列表添加自定义文本。

## 模型说明

本项目使用BERT-base-chinese预训练模型，构建了一个多任务分类模型，同时预测三个任务：
1. 是否存在AI应用
2. AI使用方式
3. AI应用类型

模型结构：
- 底层使用BERT预训练模型提取文本特征
- 三个分类头分别处理不同的分类任务
- 使用多任务学习的方式联合训练

## 注意事项

1. 首次运行时会自动下载BERT预训练模型，可能需要较长时间
2. 训练过程中会生成 `models` 目录存储模型文件
3. 评估和预测时需要先完成模型训练
4. 建议使用GPU进行训练以提高速度

## 结果说明

训练完成后，模型可以：
- 自动识别企业年报中是否提及AI应用
- 分类AI的使用方式
- 分类AI的应用类型

这些结果可以用于分析制造业企业的数字化转型程度和AI应用情况。
